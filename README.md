# WARFRONT

A time-bending puzzle adventure game built with Phaser.js and TypeScript.

## Overview

WARFRONT is a 2D puzzle-adventure game where you manipulate time across five different historical eras to solve environmental puzzles and prevent a catastrophic temporal paradox.

## Tech Stack

- **Engine:** Phaser 3
- **Language:** TypeScript
- **Platform:** Web + Android (Cordova)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
