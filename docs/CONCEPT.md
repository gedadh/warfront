# WARFRONT - Oyun Konsepti

## Tema
Tarihçi kaşif, geçmişe yolculuk, zaman makinesi ile tarihi olayları keşfetme.

## Tür
2D Puzzle-Adventure

## Hedef Kitle
12+ yaş, mobil ve web oyuncuları

## Platform
- Web (PWA)
- Mobil (Capacitor ile wrapper)

## Özellikler
- Zaman yolculuğu mekaniği
- Tarihi olaylar ve mekanlar
- Puzzle çözme
- Hikaye odaklı gameplay
